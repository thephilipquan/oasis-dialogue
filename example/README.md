# example

This example project includes the following:

* An example [OasisDialogue project](dialogue/).
* The exported [JSON](dialogue.json) contains the dialogue flow.
* The exported [CSV](dialogue.csv) that the JSON references to build dialogue in the language specified at runtime.
* An example [manager](example_dialogue_manager.gd) implementation.
* A [custom annotation](track_traverser_controller.gd) to track the number of times a branch has been visited.
* An [example](example.gd) scene that shows how to start and continue a dialogue.
